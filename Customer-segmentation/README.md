# Customer Segmentation Using Machine Learning

This project performs customer segmentation using machine learning techniques to group customers based on similar characteristics such as age, gender, and spending behavior. The implementation was developed using Python in **Google Colab**.

---

## Problem Statement

Customer segmentation is the process of dividing customers into groups based on shared characteristics. Businesses use segmentation to better understand customer behavior and create targeted marketing strategies.

The goal of this project is to apply clustering techniques to identify meaningful customer groups.

---

## Dataset

Mall Customers Dataset

Dataset Link:
[https://www.kaggle.com/shwetabh123/mall-customers](https://www.kaggle.com/shwetabh123/mall-customers)

Dataset features include:

* Customer ID
* Gender
* Age
* Annual Income
* Spending Score

These features help analyze customer purchasing patterns.

---

## Machine Learning Technique Used

### K-Means Clustering

This project uses the **K-Means clustering algorithm** to group customers into clusters based on similar characteristics.

Key points:

* Groups customers into **K clusters**
* Customers within the same cluster have similar behavior
* Helps businesses design personalized marketing strategies
