# Machine Learning Portfolio

This repository contains multiple machine learning implementations covering regression, classification, clustering and reinforcement learning.

Projects:

1. Housing Price Prediction (Regression)
2. Height / Weight Prediction
3. MNIST Digit Recognition (KNN)
4. Customer Segmentation (K-Means)
5. Parkinson’s Disease Detection (XGBoost)
6. Wine Quality Model Comparison
7. Spam Email Classification
8. Fake Review Detection (SVM)
9. Gridworld Path Optimization (Q-learning)
10. Movie Review Sentiment Analysis

Tools Used:
Python, Pandas, Scikit-learn, XGBoost, Matplotlib, Seaborn
