
# Housing Price Prediction

This project predicts housing prices using machine learning regression models.  
The implementation was developed using Python in **Google Colab**.

## Problem Statement

The goal of this project is to build a regression model that can predict the price of a house based on different features.

## Dataset
house_data.csv


## Algorithms Used

Two regression algorithms were implemented and compared:

### Linear Regression
A statistical method used to model the relationship between dependent and independent variables using a linear equation.

### Random Forest Regressor
An ensemble learning method that builds multiple decision trees and combines their predictions to improve accuracy.

- numpy
- pandas
- matplotlib
- scikit-learn
