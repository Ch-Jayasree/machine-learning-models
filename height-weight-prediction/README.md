# Height and Weight Prediction

This project predicts a person's height or weight using machine learning regression algorithms.  
The model was implemented using Python in **Google Colab**.

## Problem Statement

The objective of this project is to build a regression model that predicts height or weight based on available data features.

## Dataset
SOCR-HeightWeight.csv

## Algorithms Used

Two regression models were implemented:

### Linear Regression
A basic regression algorithm used to model the relationship between input variables and output variables.

### Random Forest Regressor
An ensemble machine learning algorithm that combines multiple decision trees to improve prediction accuracy.

## Libraries Used

- numpy
- pandas
- matplotlib
- scikit-learn

## Notebook

The implementation is available in:

Height-Weight_data.ipynb

## Workflow

1. Load dataset
2. Data preprocessing
3. Train–test split
4. Train regression models
5. Predict height or weight
6. Evaluate model performance

## How to Run

1. Open the notebook in Google Colab
2. Run all cells sequentially
3. The models will train and display prediction results
